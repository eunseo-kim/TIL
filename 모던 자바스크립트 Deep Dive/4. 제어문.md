# 📝자바스크립트의 제어문(8장) / TIL(21.03.08)
## 📌 제어문
-  **삼항 연산자** : 경우의 수가 3개(양수, 음수, 0)라면?
```js
var num = 10;
var kind = num ? (num > 0 : "양수" : "음수") : "0";
console.log(kind); // 양수
```
- **`break`문** : `break`문은 레이블 문, 반복문, `switch`문의 코드 블록 내에서만 사용할 수 있다.
- **레이블 문** : 식별자가 붙은 문

  - 주로 중첩된 for문의 내부 for문에서 내부 for문만이 아닌 외부 for문까직도 탈출하기위해 레이블 문을 사용한다.

  ```js
  // 예시) foo라는 식별자가 붙은 레이블 문
  foo : console.log("foo");
  
  // 레이블 문으로 이중 for문 탈출하기
  outer : for(var i = 0; i < 3; i++) {
      for (var j = 0; j < 3; j++) {
          if (i+j == 3) break outer; // 이중 for문을 탈출한다.
      }
  }
  ```

  - 레이블 문은 **중첩 for문을 탈출**할 때는 유용하지만, 그 밖의 경우는 **일반적으로 권장하지 않는다.** 
  - 레이블 문을 사용하면 **프로그램의 흐름이 복잡해져서 가독성이 나빠지고 오류를 발생시킬 가능성이 높아지기 때문**이다.
