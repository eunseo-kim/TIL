## 🔍네트워크 엣지

- 네트워크 엣지 = 종단 장치가 연결되는 **접속 네트워크(Access Network)**

#### ✔접속 네트워크(Access Network) 구분

- 홈 접속 네트워크(Home Access Network = Residential Network) ✅
- Enterprise Access Network ✅
- Mobile Access Network

#### ✔DSL(Digital Subscriber Line) 네트워크

- **하나의 전화선**을 사용해서 **음성 신호와 고속의 데이터 신호를 동시에 전달**하는 네트워크이다.

![img](https://t1.daumcdn.net/cfile/tistory/2330D04D58C7C9D00A)

- 종류: ADSL / VDSL

#### ✔Cable 네트워크

- **동축 케이블**에서 TV용 비디오 전송용으로 사용되고 있지 않은
  **남은 부분(상향데이터/하향데이터)을 인터넷 데이터 신호 전송용으로 사용**하는 기법

![img](https://media.vlpt.us/images/ckstn0777/post/c1a543a0-c7f8-4c49-b214-6388484cd630/image.png)

- 상향 : 1Gbps 전후의 전송 속도 **(공유 속도)**
- 하향 : 10Gbps 전후의 전송 속도 **(공유 속도)**

> **💡 DSL과의 차이점**
>
> - DSL, Cable 둘 다 [음성/비디오] 전송용을 제외한 나머지 부분을 
>   인터넷 신호 전송용으로 사용한다는 점은 동일하지만,
> - **DSL은 전용 속도를 제공하고,**
>   **Cable은 공유 속도를 제공한다.**

#### ✔FTTH(Fiber To The Home) 네트워크

- **광 케이블 하나의 전송 속도**를 여러 가정에 **공유**하는 방식이다.

- 광케이블은 구리선보다 품질이 좋아서 구리선, Cable보다 고속의 성능을 가진다.

> ##### 💡 TDM-PON (시간 분할 다중 접근)
>
> - 광 케이블에 연결된 모든 가입자가 **시간 분할 다중 접근**으로 광케이블의 대역폭을 공유한다.
>
> ##### 💡 WDM-PON (파장 분할 다중 접근)
>
> - Splitter에 여러개의 다른 파장을 전달하면
>   각 가입자마다 **서로 다른 파장의 광 신호가 할당**되어 제공된다.
> - 전송속도가 빠르다.

#### ✔Ethernet / WiFI

- **Ethernet** = **유선 LAN**을 구축하는 기술
- **WiFi** = **무선 LAN**을 구축하는 기술

#### ✔물리 매체 (Physical Media)

- 정보를 표현하는 **신호를 전달하는 매개체**
- 통신 장치의 **물리적 연결 매개체**

> **💡 유도 매체(유선)**
>
> - TP(구리선 2개 꼬아서 만듦), 동축 케이블, 광케이블
>
> **💡 비유도 매체(무선)**
>
> - WiFi, 위성, 이동전화 무선 채널
